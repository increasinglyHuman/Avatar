# BlackBox Avatar

Web-based avatar customization tool for [poqpoq World](https://poqpoq.com/world/).

Part of the **BlackBox Creative Suite**:
- [Animator](https://poqpoq.com/animator/) — GLB animation editor with IK
- [Skinner](https://poqpoq.com/skinner/) — Vertex weight painter
- [Legacy](https://poqpoq.com/legacy/) — OAR/OpenSim converter
- [Landscaper](https://github.com/increasinglyHuman/Landscaper) — Procedural world population
- **Avatar** — Character customization (you are here)

## Status

Early development. Babylon.js scaffold in progress.
